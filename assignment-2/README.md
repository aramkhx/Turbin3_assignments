## 🎯 Assignment Objectives

The primary goal of this assignment was to gain hands-on experience with Solana's native token standards and the Metaplex (MPL) Core program. 

- [x] Mint a custom SPL token.
- [x] Mint an NFT using MPL Core.
- [x] Use any one of the core plugins available in MPL Core for the NFT.
