# 🚀 Assignment 1: Guessing Game

This is the second assignment for the **Turbin3 Solana Builder Bootcamp**. The goal was to build the Guessing Game project from the Rust Book.

## 📌 Overview

This is a simple command-line game where:

- The program generates a random number
- The user tries to guess the number
- The program provides feedback:
  - Too small
  - Too big
  - Correct 🎉
