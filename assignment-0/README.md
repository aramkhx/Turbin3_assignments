# 🚀 Assignment 0: Program Initialization

This is the first assignment for the **Turbin3 Solana Builder Bootcamp**. The goal was to set up the development environment and initialize a basic Solana program using the Anchor framework.

## 📝 Features

- **Anchor Framework**: Initialized using `anchor init`.
- **Program Structure**: Standard Anchor program layout with `programs`, `tests`, and `migrations`.
- **Local Testing**: Configured for local validator testing.

## 🛠️ Commands

### Build
```bash
anchor build
```

### Test
```bash
anchor test
```

### Deploy (Localhost)
```bash
solana-test-validator
anchor deploy
```
